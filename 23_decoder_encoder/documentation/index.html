Overview

Encode & Decode Online is very powerful tool to your Business. This is a readymade script to convert Text/Link for your Visitor & Customer to rapid increase of your Business. No need of database. Good Ads Spaces are available.

Security : It is protected against SQL injection, XSS and other attack and your website will be safe.

Requirements

Before starting to install, make sure you fulfill the following requirements:
  • For Local Server installation, you need to have XAMPP, WAMP or any apache server with PHP and MySQL
  • You can use this script on any live LINUX server.
  • Required PHP version in server >= 5.4.

Features

  • Convert Text/Link into Encoded Method.
  • Convert Text/Link into Decoded Method.
  • Click to Copy Function
  • Reset Whole Text/Link on 1 Click.
  • No Page Reload
  • Ads Space
  • Company Name & Link in Footer
  • No Database Needed

How to Upload Script ?

  • Step:1 = Go to Downloaded folder which you have purchased named decoder.zip
  • Step:2 = Unzip decoder.zip

  • Step:3 = Open decoder folder and go to upload folder.

  • Step:4 = After opening upload folder, copy all the files and upload to your server root folder (www directory) or inside any subfolder.

  • Example 1: Lets assume you have uploaded all the files and folders inside your root directory i.e. www folder then your website URL for this script is https://www.yourwebsite.com/
  • Example 2: Lets assume you create a folder decoder inside your root directory and you have uploaded all the files and folders inside your subfolder e.g. taxcalculator then your website URL for this script is https://www.yourwebsite.com/decoder

1) If you do not want to show any ads.

  • Step:1 = Go to upload folder and open index.php in any text editor like Notepad++, Dreamweaver, etc.
